⸻

Stripe System

The stripe graphic is an official part of the primary brand logo.

Stripe Structure

The stripe graphic contains exactly 3 stripes:
	1.	Purple
	2.	White
	3.	Green

Rules:
	•	Exactly three stripes
	•	Equal spacing between stripes
	•	No gradients
	•	No highlights inside stripes
	•	No internal split inside green stripe
	•	Smooth racing curve
	•	Transparent background when exported separately

⸻

Stripe Usage Rules

Use logo with stripe for:
	•	Website header logo
	•	Website footer logo
	•	Email signature logo
	•	Main brand presentations
	•	Marketing banners
	•	YouTube banners / channel art
	•	Promo graphics
	•	Brand identity placements

Use logo without stripe for:
	•	Watermarks on caption images
	•	Monochrome editorial overlays
	•	Small placements where stripe becomes too detailed
	•	Utility/logo stamp usage

Use HQ icon for:
	•	Instagram profile
	•	Facebook profile
	•	Threads profile
	•	Twitter/X profile
	•	BlueSky profile
	•	Favicon
	•	Apple touch icon
	•	YouTube profile icon
